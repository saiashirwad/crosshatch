# Quickstart

## Overview

Crosshatch is an embedded wallet that turns standard x402 payment requirements into signed payment payloads, thereby
enabling seamless payment to x402 merchant.

## Installation

:::code-group

```bash [npm]
npm install crosshatch
```

```bash [pnpm]
pnpm add crosshatch
```

```bash [bun]
bun add crosshatch
```

:::
