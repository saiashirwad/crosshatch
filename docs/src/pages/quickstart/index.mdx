# Quickstart

Crosshatch is a general purpose Effect toolkit for composing x402 payments.

## Installation

:::code-group

```bash [npm]
npm install crosshatch
```

```bash [pnpm]
pnpm add crosshatch
```

```bash [bun]
bun add crosshatch
```

:::

[GitHub issues](https://github.com/crosshatch/crosshatch/issues/new) and we'll
