# Allowances

Allowances are app-scoped spend permissions. They let a user authorize an app to create payments from the user's wallet
without approving every individual payment.

:::info[App-scoped spending] The app can spend from this allowance on behalf of the user without additional approvals.
:::

Allowances are what make autonomous paid actions possible: agents, generative UI, background tasks, and other app logic
can pay for resources as long as the request fits inside the user's approved limit.

:::tip[Revocable] Allowances can be rescinded anytime at [crosshatch.dev/rescind](https://x.crosshatch.dev/rescind). :::
