# Quickstart

## Overview

Crosshatch is an embedded wallet that turns standard x402 payment requirements into signed payment payloads, thereby
enabling seamless payment to x402 merchant.

## Installation

:::code-group

```bash [npm]
npm install crosshatch
```

```bash [pnpm]
pnpm add crosshatch
```

```bash [bun]
bun add crosshatch
```

:::

## Core Functionality

Whenever your app requires a payment payload, `propose` it:

```ts
import { Effect } from "effect"
import { propose, CrosshatchRuntime, settle } from "crosshatch" // [!code focus]

Effect.gen(function* () {
  const { payload } = yield* propose({ required }) // [!code focus]
  yield* settle({ payload })
}).pipe(CrosshatchRuntime.runPromise)
```

A user's first proposal will open a secure dialog to create, link and/or fund a Crosshatch wallet. Thereafter, payloads
can be created without additional prompts while the payment is within the app's allowance.

## Settle Payment

```ts
import { Effect } from "effect"
import { propose, CrosshatchRuntime, settle } from "crosshatch" // [!code focus]

Effect.gen(function* () {
  const { payload } = yield* propose({ required })
  yield* settle({ payload }) // [!code focus]
}).pipe(CrosshatchRuntime.runPromise)
```

## Settle Hundreds of Payments

```ts
import { Effect } from "effect"
import { propose, CrosshatchRuntime, settle } from "crosshatch"

Effect.gen(function* () {
  // [!code focus:start]
  for (let i = 0; i < 1000; i++) {
    const { payload } = yield* propose({ required })
    yield* settle({ payload })
  }
  // [!code focus:end]
}).pipe(CrosshatchRuntime.runPromise)
```

## Fetch X402 Resources

Crosshatch's `fetch` wraps `globalThis.fetch` so that 402s from any x402-compliant merchant are intercepted and retried
automatically.

```ts
import { fetch } from "crosshatch"

const response = await fetch("https://merchant.example/resource")
```
